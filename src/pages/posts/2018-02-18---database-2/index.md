# 数据库学习（二）(待续)


### 子查询

表达式表示一个计算结果，子查询的结果仍然是一个关系，
两个关系做笛卡尔积，之后从中选取满足一定条件的子集，就叫做连接。
连接两个表的操作，会被用在

- 等值连接查询
- 自然连接查询
- 非等值连接查询
- 自身连接查询
- 外连接查询
- 复合条件连接查询

## 等值连接查询

等值连接和非等值连接是对立的。对两张表做笛卡尔积后，选择其中某些属性相同的元祖，称为等秩连接。

等值连接之后

```sql
select * from course,
```
